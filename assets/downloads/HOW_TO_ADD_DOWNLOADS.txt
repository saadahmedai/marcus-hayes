HOW TO CREATE EACH FILE:
Use Canva (free) to design each PDF.
  1. Go to canva.com
  2. Create new design → choose "A4 Document"
  3. Design the worksheet/cheat sheet
  4. Download as PDF
  5. Rename the file to match the exact name above
  6. Place it in this folder (assets/downloads/)

After adding a PDF, find the matching download
button in the course HTML file and update:
  onclick="alert('...')"
  to:
  href="assets/downloads/FILENAME.pdf"
  download

Example:
  BEFORE: onclick="alert('Add your PDF...')"
  AFTER:  href="assets/downloads/niche-worksheet.pdf" download

HOW TO UPDATE THE BUTTON:
Open the course HTML file in any text editor.
Search for the PDF filename (e.g. niche-worksheet.pdf)
Replace the onclick with href + download as above.
